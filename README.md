# Archong Liu

I'm a Cloud Engineer in Taipei. I spend most of my time figuring out why cloud bills are so high — and then making them not. At Futurnest I cut AWS compute costs by 80%+ through a mix of Fargate Spot, ARM64 migration, and right-sizing. I also care a lot about infrastructure being reproducible, so I've been deep in IaC land (Terraform, CDK) for a while.

Outside of work I write about what I learn. I've put out 30+ DevOps and cloud articles on [IThome](https://ithelp.ithome.com.tw/) and spoke at AWS Community Day Taiwan 2025 on building IaC infrastructure. If something took me a week to figure out, I try to make sure the next person figures it out in an hour.
